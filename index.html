Mes projets réalisés

Convertisseur de bases

Projet sous Linux en langage C réalisé avec l'éditeur de code Geany qui consiste à convertir en hexadécimal et en binaire les nombres entrés par l'utilisateur en base 10.
Lors de ce projet tout les scripts .c et .h étaient compilés et exécutés dans le terminal Linux en bash.

Dates : Décembre 2020 - janvier 2021.

Tower Defense sous Unity

Projet sous Unity et Visual Studio en C# qui consiste à créer en 3 semaines un Tower Defense. Ce qui implique: la gestion des ennemis (leurs déplacements, leurs points de vie et leur fréquence d'apparition, les dégâts qu'ils infligent,...), la gestion des tourelles de défense (dégâts infligés aux ennemis, fréquence de tir, lieu de génération,...) et des écrans de jeu (accueil, crédits, pause, victoire et défaite).

Dates : Octobre-Novembre 2021

API alimentaire

Projet sous Visual Studio Code en HTML5 et Java Script qui consiste à créer une API alimentaire. C'est-à-dire, lorsque l'utilisateur entre l'EAN (code produit) d'un produit, la page internet lui renvoie une image de ce produit, sa marque, son nom, sa composition et son nutriscore.
La page Internet va récupérer les informations depuis le .json du produit répertorié dans la base publique OpenFoodFact.

Dates : Novembre - Decembre 2021

Création et interrogation d'une base de données

Projet SQL qui consiste à créer une petite base de données avec Wampserver64 et l'interroger en SQL pour en extraire des informations ciblées.

Dates : Decembre 2021
